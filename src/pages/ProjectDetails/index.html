Overview

This project uses statistical and machine learning techniques to predict tsunami impact using historical data. By analyzing patterns in seismic and oceanic parameters, the model aims to enhance early warning systems.

About the Project

Problem Statement: Traditional models struggle to predict the complex impacts of tsunamis. This project aims to bridge that gap using data-driven machine learning methods.

Objectives: Predict tsunami wave height, affected population, and damages using seismic and oceanographic features.

Real-world Impact: Helps disaster management authorities improve response strategies and mitigate losses.

Dataset & Sources

We used the NOAA NGDC Historical Tsunami Event Database and a cleaned Kaggle dataset.

  • Records: 1500+ tsunami events
  • Features: Date, Magnitude, Wave Height, Deaths, Country, Coordinates

View Kaggle Dataset

Methodology

Preprocessing: Missing value imputation, normalization (Min-Max, Z-score)

Feature Engineering: PCA for seismic features, derived features like normalized wave height

Models: Decision Tree, Random Forest, SVM, Neural Networks
